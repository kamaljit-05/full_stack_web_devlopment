
HTML COMPLETE NOTES (DETAILED)

1. INTRODUCTION TO HTML
Definition:
HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages. It tells the browser how content should be displayed.

Features:
- Easy to learn
- Platform independent
- Supports multimedia
- Works with CSS and JavaScript
- Foundation of web development

Example:
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

2. HTML DOCUMENT STRUCTURE

<!DOCTYPE html>  -> Declares HTML5
<html>           -> Root element
<head>           -> Metadata section
<title>          -> Browser tab title
<body>           -> Visible webpage content

27. INTERVIEW QUESTIONS

Q1. What is HTML?
Ans. HyperText Markup Language used to create web pages.

Q2. Difference between HTML and CSS?
Ans. HTML structures content while CSS styles it.

Q3. What is a semantic tag?
Ans. A tag that describes the meaning of content.

Q4. Difference between div and span?
Ans. Div is block-level; span is inline.

Q5. What is the purpose of the alt attribute?
Ans. Provides alternative text for images.
